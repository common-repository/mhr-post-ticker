== Description ==

This plugin will enable a news ticker in your WordPress theme. You can use news ticker via shortcode everywhere you want, even in theme files.

== Installation ==

1. Upload `mhr-post-ticker.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place this shortcode [mhr_post_ticker] to any page, post or in widgets
4. Place `<?php echo do_shortcode("[mhr_post_ticker]"); ?>` in your templates file
